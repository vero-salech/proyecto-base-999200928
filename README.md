# Proyecto base: Maquetación UI clon WhatsApp — Entrega

## Descripción
Realicé en este proyecto la maquetación en HTML y CSS de una interfaz similar a WhatsApp.

Reproduje la estructura general de la imagen propuesta, que incluye un sidebar izquierdo con la lista de chats y un panel derecho con la conversación activa. 

Para el desarrollo utilicé únicamente HTML y CSS.

## Estructura del proyecto
/index.html  
/styles.css   

## Cómo visualizar el proyecto
Abrir el archivo `index.html` en cualquier navegador web.

## Decisiones de diseño y consideraciones
Si bien utilicé colores diferentes a los de la interfaz original, elegí la tipografía con un estilo similar.
Utlicé Flexbox para la distribución en columnas y la alineación de los elementos.
Los estilos los organicé en un único archivo CSS.
